Field Examples

Dropdown

This dropdown allows you to freely style your dropdown,

Select a Nut
Almonds
Cashews
Pistachios
Walnuts
Pecans
Hazlenuts

instead of using this ugly html select tag.

Setting Up
HTML Code

You may make up the ID however you wish, but you can't change any class names. The data-value="-1" indicates the default state.

 
<div id="my-dropdown" class="tc-field tc-dropdown">
	<div class="display"></div>
	<div class="menu">
		<div data-value="-1" class="item state-empty">Select a Nut</div>
		<div data-value="1" class="item">Almonds</div>
		<div data-value="2" class="item">Cashews</div>
		<div data-value="3" class="item">Pistachios</div>
		<div data-value="4" class="item">Walnuts</div>
		<div data-value="5" class="item">Pecans</div>
		<div data-value="6" class="item">Hazlenuts</div>
	</div>
</div>
JavaScript Code

The element: takes the id of the dropdown you have set up. Instead of "#my-dropdown" you can also write, $('#my-dropdown'). We are using jQuery to fetch html elements.

 
var dropdown = NI.field.new_instance({
	element: "#my-dropdown", 
	options: {
		handlers: {
			change: function(e, d) {
				if (d) { 
					console.info("change from "+ d.from +" to "+ d.to);
				}
			}
		}
	}
});

Dropdown-Not using html

This dropdown is a same dropdown as above, but you don't need any HTML code. How awesome is that.

Setting Up
JavaScript Code

This one is pretty self explanatory, you would increment numeric values along with matching order of text values. So, as an example, if you wish to add third element to your drop down, in the value_list, you would add "2": "C Train". Remember to add commas to previous elements.

 
var headlessDropdown = new NI.field.Dropdown({
	value_list: {
		"0": "A Train",
		"1": "B Train"
	},

Also, you can tweak this empty_text value as well.

 
	empty_text: "Select a Train"
});

".headless" is a class name of a div in HTML. You can create a div with a specific class name to designate your Dropdown or you don't have to make one and just append to the DOM.

 
headlessDropdown.append_to(".headless");

MultiDropdown

MultiDropdown allows you to select multiple values on a dropdown.

The values that you choose are highlighted in red and checked.

Select a Train
A,B,C Train
N,Q,R Train
B,D,F,M Train
1,2,3 Train
4,5,6 Train
7 Train
Setting Up
HTML code

Tbe only difference with this dropdown is that you add "tc-multiple" in classes.

 
<div id="my-multidropdown" class="tc-field tc-dropdown tc-multiple">
	<div class="display"></div>
	<div class="menu">
		<div data-value="-1" class="item state-empty">Select a Train</div>

You can take away <input type="checkbox"> if you don't want to see check boxes, the red highlighting will still allow you to distinguish what are selected. 

		<div data-value="1" class="item"><input type="checkbox">A,B,C Train</div>
		<div data-value="2" class="item"><input type="checkbox">N,Q,R Train</div>
		<div data-value="3" class="item"><input type="checkbox">B,D,F,M Train</div>
		<div data-value="4" class="item"><input type="checkbox">1,2,3 Train</div>
		<div data-value="5" class="item"><input type="checkbox">4,5,6 Train</div>
		<div data-value="6" class="item"><input type="checkbox">7 Train</div>
	</div>
</div>
JavaScript Code

Nothing changed from the JavaScript set up for regular Dropdown.

 
var multiDropdown = NI.field.new_instance({
	element: "#my-multidropdown",
	options: {
		handlers: {
			change: function(e, d) {
				if (d) { 
					console.info("change from "+ d.from +" to "+ d.to);
				}
			}
		}
	}
});

Input Field-with Validator

This input field has validator built in it.

On a following example, try inputting values other than numbers. The field area will be highlighted in red.

Setting Up
HTML Code

This does not need much of a set up in HTML.

 
<div class="tc-field">
	<input id="my-input" type="text">
</div>
JavaScript Code
var input = NI.field.new_instance({
	element: "#my-input",
	options: {

This is where you get to include extentions/functionalities that you may want to include in your input field. For the example, there are Validator and Hint. For more explanations on Validators check Validators tab.

 
		extensions: {
			Validator: {

Here, we are indicating that the input kind will be number and nothing else.

 
				validators: ["number"]
			},
			Hint: {

This is where you get to tweak hint message, and its placement according to your field size with css.

The css here is referencing to the left top corner of your field box.

 
				content: "Enter a number",
				css: {
					top: 5,
					left: 5
				}
			}
		},
		handlers: {

These handlers are included for you to perhaps in future bind functionalities, in each handling functions.

 
			focus: function(e) {
				console.info("focus");
			},
			blur: function(e) {
				console.info("blur");
			}
		},

You can put in explicit values if you wish. We left this blank to display the hint.

 
		val: ""
	}
});

Textarea

There is nothing too fancy in this textarea field, except you can add your hint just like the input field.

This is using the regular html textarea tag, but with our JavaScript, it allows you to add in functionalities (Hint for now).

Setting Up
HTML Code

Simple as that...

 
<textarea id="my-textarea"></textarea>
JavaScript Code
var textarea = NI.field.new_instance({
	element: "#my-textarea",
	options: {
		extensions: {

As exaplained in Input field, you can tweak the hint message here and placement of the message with css top and left property.

 
			Hint: {
				content: "Enter a Comment",
				css: {
					top: 5,
					left: 5
				}
			}
		}
	}
});
